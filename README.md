# Cookenu :shallow_pan_of_food:	

Projeto backend desenvolvido no bootcamp da Labenu. \
Cookenu é uma API REST para criar receitas e conhecer pessoas que gostam de comida! :woman_cook:

1- Clone este repositório

```

```
npm install
